1: How dose javascript work ?

Ans : জাভাস্ক্রিপ্ট হল ওয়েবের জন্য একটি স্ক্রিপ্টিং ভাষা। Html এবং css এর মাধ্যমে web page টা দেখানো হয় এবং এই web page টার মধ্যে কোনো ঘটনা ঘটানো অর্থাৎ inter active কাজ করে এই javascript. যেমন : কোনো একটা তে click করলে কোনো একটা জিনিস change হবে। html যেমন ওয়েব পেইজ এর কাঠামো তৈরি করে এবং css যেমন এই কাঠামো কে style করে সুন্দর করে। তেমনই javascript inter activity বা বিভিন্ন ধরনের কাজ কারবার করে থাকে। DOM (Document Object Model)জাভাস্ক্রিপ্ট কোডে ওয়েব পৃষ্ঠার একটি লাইভ ভিউ উপস্থাপন করে।

2: Local storage and session storage differences ?

Ans :

Local storage session storage
▸ডাটা নিজে হাতে delete বা clear করতে হয়। ▸ব্রাউজার অথবা ট্যাব বন্ধ করলে storage clear হয়ে যায়।
▸কোন নির্দিষ্ট ট্যাবের জন্য কাজ না করে যেকোন ট্যাবের জন্য কাজ করে। ▸কোন নির্দিষ্ট ট্যাবের জন্য কাজ করে আর ট্যাব বন্ধ করলে ডাটা মুছে যায়।
▸উদাহরণ : আমরা যেমন Google account log in করার পর আবার যখন ওয়েব ব্রাউজারে করি আবার পুনরায় log in করতে হয় না। এইটা local storage এ থাকে। ▸উদাহরণ : আমরা যখন you tube এ video দেখি সময় যদি ওয়েব ব্রাউজার টা বন্ধ করে দেই। তার পর যদি পুনরায় ব্রাউজ করি তখন র আগের ভিডিও আসে না। অর্থাৎ session শেষ হয়ে গেলে session storage এর ডাটা ও clear হয়ে যায়।

3: What dose javascript event loop do ?

Ans : javascript event loop প্রোগ্রাম সম্পর্কে যুক্তি করার সময় কিছু চমৎকার বৈশিষ্ট্য অফার করে, যার মধ্যে রয়েছে যে যখনই একটি ফাংশন চলে, এটিকে পূর্বনির্ধারিত করা যাবে না। এবং অন্য কোনও কোড চালানোর আগে এটি সম্পূর্ণরূপে চলবে (এবং ফাংশনটি ম্যানিপুলেট করে ডেটা পরিবর্তন করতে পারে)। এটি C থেকে ভিন্ন, উদাহরণস্বরূপ, যেখানে একটি ফাংশন যদি একটি থ্রেডে চলে, তবে অন্য থ্রেডে অন্য কিছু কোড চালানোর জন্য রানটাইম সিস্টেম দ্বারা এটি যে কোনো সময়ে বন্ধ হয়ে যেতে পারে। এই মডেলের একটি নেতিবাচক দিক হল যে যদি একটি বার্তা সম্পূর্ণ হতে খুব বেশি সময় নেয়, ওয়েব অ্যাপ্লিকেশনটি ক্লিক বা স্ক্রোল করার মতো ব্যবহারকারীর মিথস্ক্রিয়া প্রক্রিয়া করতে অক্ষম। ব্রাউজার "একটি স্ক্রিপ্ট চালানোর জন্য খুব বেশি সময় নিচ্ছে" ডায়ালগ দিয়ে এটিকে প্রশমিত করে। অনুসরণ করার একটি ভাল অভ্যাস হল বার্তা প্রক্রিয়াকরণকে সংক্ষিপ্ত করা এবং সম্ভব হলে একটি বার্তাকে কয়েকটি বার্তায় কেটে ফেলা।

1: How dose javascript work ?

Ans : জাভাস্ক্রিপ্ট হল ওয়েবের জন্য একটি স্ক্রিপ্টিং ভাষা। ওয়েব ব্রাউজার একটি ওয়েব পেজ লোড করে, HTML পার্স করে এবং যা থেকে ডকুমেন্ট অবজেক্ট মডেল (DOM) নামে পরিচিত তা তৈরি করে। DOM জাভাস্ক্রিপ্ট কোডে ওয়েব পৃষ্ঠার একটি লাইভ ভিউ উপস্থাপন করে। HTML এবং CSS রাখা হয় DOM দ্বারা একসাথে প্রথমে ওয়েব পেজ তৈরি করুন। তারপর, ব্রাউজারের জাভাস্ক্রিপ্ট ইঞ্জিন জাভাস্ক্রিপ্ট ফাইল লোড করে এবং ইনলাইন কোড কিন্তু অবিলম্বে কোড চালায় না। এটি HTML এবং CSS শেষ হওয়ার জন্য অপেক্ষা করে লোড হচ্ছে
